# 常用漢字表は専門分野や個別事情まで一律に縛る趣旨ではない

## 要点

文化庁の国語施策では、常用漢字表は一般社会生活の目安であり、専門分野や個別事情を一律に縛るものではないという趣旨で運用されている。

## 主張

常用漢字表は、専門分野や個別事情まで一律に縛る趣旨のものではない。
